# 医疗 Agentic RAG 测试用例

## 第一类：知识库精确检索（应命中本地指南）

### TC-01 肝癌高危人群与筛查

```
Q: 原发性肝癌的高危人群有哪些？推荐的筛查方案是什么？
验证: 应引用《原发性肝癌诊疗指南（2024年版）》第二章
      → 高危人群：HBV/HCV感染、过度饮酒、肝脂肪变性、肝硬化、家族史
      → 筛查：超声显像 + 血清AFP，至少每6个月1次
```

### TC-02 儿童肺炎支原体肺炎用药

```
Q: 儿童肺炎支原体肺炎的首选抗生素是什么？
验证: 应引用《儿童肺炎支原体肺炎诊疗指南（2025年版）》
      → 大环内酯类抗生素（阿奇霉素等）
```

### TC-03 黑色素瘤 ABCDE 法则

```
Q: 黑色素瘤的早期识别有哪些特征？
验证: 应引用《黑色素瘤诊疗指南（2022年版）》
      → ABCDE法则：非对称、边缘不规则、颜色改变、直径>5mm、隆起
```

### TC-04 流感诊疗方案

```
Q: 2025年流行性感冒的推荐抗病毒药物有哪些？
验证: 应引用《流行性感冒诊疗方案（2025版）》
      → 神经氨酸酶抑制剂（奥司他韦等）、RNA聚合酶抑制剂
```

### TC-05 肥胖症诊断标准

```
Q: 肥胖症的诊断标准是什么？
验证: 应引用《肥胖症诊疗指南（2024年版）》
      → BMI ≥ 28 kg/m² 为肥胖
```

## 第二类：跨文档对比推理（需融合多份指南）

### TC-06 多癌种治疗对比

```
Q: 比较肝癌和胰腺癌的五年生存率和主要治疗手段
验证: 应同时引用《原发性肝癌诊疗指南（2024年版）》和
      《胰腺癌诊治指南（2022年版）》
```

### TC-07 涉及免疫治疗的癌症

```
Q: 有哪些癌症诊疗指南提到了免疫治疗（PD-1/PD-L1 抑制剂）？
验证: 应全文检索所有指南，筛选免疫治疗相关内容
```

### TC-08 筛查年龄建议对比

```
Q: 各癌症指南中推荐的筛查起始年龄有什么不同？
验证: 应综合多份指南，对比不同癌症的筛查起始年龄
```

## 第三类：联网补充（本地无直接答案）

### TC-09 最新药物批准信息

```
Q: 2025年FDA新批准了哪些抗癌药物？
验证: 本地指南无此信息 → 应自动调用 web_search 联网检索
      → 标注来源和时效性
```

### TC-10 非诊疗类咨询

```
Q: 中国最好的肿瘤医院有哪些？
验证: 指南中不含医院排名 → 应联网搜索
      → 标注信息来源
```

## 验收标准

|   类别   | 测试数 | 最低准确率 | 验证方法                                   |
| :------: | :----: | :--------: | ------------------------------------------ |
| 精确检索 |   5    |   ≥ 90%    | 检查引用指南名称、章节、证据等级是否正确   |
| 对比推理 |   3    |   ≥ 80%    | 检查是否引用了多份指南，融合逻辑是否合理   |
| 联网补充 |   2    |   ≥ 70%    | 检查是否正确判断需要联网，答案是否标注来源 |
